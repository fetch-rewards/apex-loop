# Apex Loop

Portable showcase artifacts for the Apex Loop project, built by Team 175 for Innovation Week 2026.

Apex Loop is a self-improvement loop that keeps AI agents good over time: an eight-stage cycle that runs mostly automated, with a human kept at the one decision that matters. This repository holds the standalone deck, interactive, and report so they can be shared and viewed on their own. The actual implementation lives in the consumer-agent repository, not here.

## Viewing

This repo is served via GitHub Pages (Fetch org members only, behind SSO). Open `index.html` for the landing page, which links to the interactive, deck, and report.

## Files

- `index.html` - landing page linking to the three artifacts below
- `apex_loop_interactive.html` - the eight-stage loop with a Today-manual / Vision-automated toggle
- `apex_loop_deck.html` - the pitch deck (Innovation, Execution, Impact)
- `apex_loop_report.html` - the full write-up: the general loop, the use case, the run, and findings
- `DECK_CONTENT.md` - source content for the deck
- `USE_CASE_REPORT_ai_assistant.md` - the AI Assistant use-case report
- `SUMMARY.md` - project summary
